# Amnis
The main repository for all the code for CMPS 115.

Setup instructions:
1. cd into 'Amnis' directory.
2. Run 'npm install' and 'npm run client-install'.
3. Install Redux Devtools Extension for Chrome.
4. Make a 'config' directory in the Amnis directory, and copy the 'keys.js' file into config.
5. Run 'npm run dev'.
